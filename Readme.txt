Compilation and Running:
=======================

        * Dependencies
		- Python version : 3.7 or Later
		- Install the requirements.txt 
            ` pip install -r requirements.txt`
            
	* Running the code 
		$ python mydig [Domain Name] [Resource Record type]
        * Examples
		$ python mydig.py google.com A
